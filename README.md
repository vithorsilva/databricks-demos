# databricks-demos (Vithor)
Neste material, você encontrará scripts a serem executados no ecossistema Databricks, tendo como objetivo explorar a solução desde questões básicas a avançadas.

Sobre os arquivos:
* [databricks-sqlfundamental-demos](./sql/databricks-sqlfundamental-demos.ipynb): apresenta um roteiro do básico a avançado no uso da linguagem SQL para consulta a dados;
* [Dashboard - AI/BI - Meu primeiro Dashboard](dashboard/demo-meuprimeirodashboard): apresenta um roteiro para construção do seu primeiro Dashboard no Databricks;
* Roteiro para gráficos:
  * [Visualization types](https://learn.microsoft.com/en-us/azure/databricks/visualizations/visualization-types);
